# Course Generator

A complete workflow for creating vocational training courses with engaging storytelling frameworks and automated student handout generation.

---

## Overview

This repository contains two powerful systems:

1. **Student Handout Generator** - Transform framework files into professional, dark-themed PDF handouts with AI-generated questions
2. **Course Design & Storytelling Framework** - Create structured courses with learning outcomes and engaging narratives

---

## Student Handout Generator

### Features

- **Automated PDF Generation** - Transform framework markdown files into professional handouts
- **AI-Powered Questions** - Contextual self-study questions generated by Claude AI
- **Mandatory Review Workflow** - Interactive terminal interface to review/edit/approve questions
- **Dark Theme Design** - Modern, high-contrast design optimized for readability (GitHub Dark inspired)
- **Image Processing** - Automatic embedding with intelligent placeholders
- **Batch Processing** - Generate handouts for entire courses
- **Question Management** - Staging, review, and approval workflow
- **Progress Tracking** - Real-time progress indicators during generation

### Quick Start

Generate your first handout in 5 minutes!

```bash
# 1. Generate questions (first time)
python3 generate_handout.py --input frameworks/module-0-framework-REVISED.md

# 2. Review AI-generated questions
python3 generate_handout.py --review-questions module-0

# 3. Generate handout with approved questions
python3 generate_handout.py --input frameworks/module-0-framework-REVISED.md
```

### Usage

#### Single Module Generation

```bash
# Basic generation
python3 generate_handout.py --input frameworks/module-0-framework-REVISED.md

# With custom output directory
python3 generate_handout.py \
  --input frameworks/module-0.md \
  --output ~/Documents/handouts

# Dry run (validate only)
python3 generate_handout.py \
  --input frameworks/module-0.md \
  --dry-run

# Verbose logging
python3 generate_handout.py \
  --input frameworks/module-0.md \
  --verbose
```

#### Question Review

```bash
# Review AI-generated questions
python3 generate_handout.py --review-questions module-0

# Check review status for all modules
python3 generate_handout.py --review-status --course ml-for-engineers
```

#### Batch Processing

```bash
# Process all modules in course
python3 generate_handout.py --batch --course ml-for-engineers

# Process specific modules
python3 generate_handout.py \
  --batch \
  --course ml-for-engineers \
  --modules 0,1,2
```

#### Advanced Options

```bash
# Use approved questions (skip AI generation)
python3 generate_handout.py \
  --input frameworks/module-0.md \
  --use-approved-questions

# Force generation with unreviewed questions (adds warning)
python3 generate_handout.py \
  --input frameworks/module-0.md \
  --force-unreviewed

# Regenerate questions
python3 generate_handout.py \
  --input frameworks/module-0.md \
  --regenerate-questions

# Multiple formats
python3 generate_handout.py \
  --input frameworks/module-0.md \
  --formats pdf,html
```

See [USAGE_GUIDE.md](USAGE_GUIDE.md) for comprehensive usage documentation.

### Output

Generated handouts include:

- **Table of Contents** - Hyperlinked with page numbers
- **Clean Content** - Instructor sections removed
- **Self-Study Questions** - 5-10 contextual questions per section
- **Images** - Embedded or with dark-themed placeholders
- **Professional Styling** - Dark theme (#0D1117 background, #E6EDF3 text)
- **Headers/Footers** - Module title and page numbers

### Configuration

Edit `handout_config.yaml` to customize:

```yaml
global:
  output_directory: "./handouts"
  page_size: "A4"
  fonts:
    body: "Inter"
    code: "JetBrains Mono"

branding:
  theme: "dark"
  colors:
    background_primary: "#0D1117"
    text_primary: "#E6EDF3"

questions:
  enabled: true
  questions_per_section: 7
  require_review: true
  ai_generation:
    model: "claude-sonnet-4-20250514"
```

### Architecture

```
Framework MD → Parse → Transform → Generate Questions (AI)
                                           ↓
                                    Review (Manual)
                                           ↓
                        ← Approved Questions
                        ↓
              Combine: Content + Questions + Images
                        ↓
                  Layout (HTML + CSS)
                        ↓
                  Render PDF (WeasyPrint)
                        ↓
                  Dark-Themed Handout
```

## Course Design & Storytelling Framework

A complete workflow for creating vocational training courses with engaging storytelling frameworks.

### What It Does

Two powerful Claude Code skills that work together to:

1. **Design courses** with proper learning outcomes hierarchy (Course → Module → Session outcomes)
2. **Generate engaging story frameworks** for each learning outcome to create compelling educational content

### Skills Included

#### 1. Course Design Skill
Located in: `.claude/skills/course-design-skill/`

**Features:**
- Conversational course design process
- Three-level learning outcomes hierarchy (CO → MO → SO)
- Excel template generation for course import systems
- Instructional design principles (Bloom's Taxonomy, Backward Design, Gagné's 9 Events)
- Domain-specific guidance (Automotive, IT, Manufacturing)

#### 2. Storytelling Framework Generator
Located in: `.claude/skills/storytelling-framework-generator.md`

**Features:**
- 13 storytelling techniques including **Teacher-Led Story Arc**
- Audience personalization (Seniors, Adults, Juveniles)
- Cultural customization with placeholders
- Multiple output formats (Video, PPT, Interactive)
- Module-level story arcs

### Complete Workflow

#### Step 1: Design Your Course

```bash
# In Claude Code, use:
Use the course-design skill to create a course for [your topic]
```

**Example:**
```
Help me design a 5-day automotive ECU diagnostics training course
```

**Output:**
- `courses/{course-name}/00-requirements.md` - All gathered requirements
- `courses/{course-name}/01-checklist.md` - Progress tracking
- `courses/{course-name}/03-template.xlsx` - Excel template with learning outcomes

#### Step 2: Generate Story Frameworks

```bash
# In Claude Code, use:
Use the storytelling-framework-generator skill to create story frameworks from my Excel template
```

**Output:**
- `storytelling-outputs/{COURSE-CODE}-{TIMESTAMP}/`
  - `requirements.md` - Story generation requirements
  - `index.md` - Navigation file
  - `Module-X-Lesson-Y.md` - Story frameworks for each lesson

### Teacher-Led Story Arc Feature

**NEW:** First-person narrative mode where you (the teacher) deliver lectures incorporating storytelling fundamentals.

**Benefits:**
- Personal connection with students
- Builds trust and authority
- Real teacher experiences vs. fictional characters
- Particularly effective for senior/non-tech audiences

**How to use:**
During storytelling requirements gathering, choose "Teacher-Led Story Arc" for lessons where you want to share personal experiences or guide students collaboratively.

### Tips for Best Results

#### For Course Design:
- Be specific about your target learners
- Think about real-world job tasks
- Consider hands-on practice vs. theory ratio
- Include safety requirements upfront

#### For Storytelling:
- Let the skill select techniques (it uses rotation)
- Provide cultural context for better customization
- Use Teacher-Led Arc for first lessons to build rapport
- Specify output format (video/PPT) for better adaptations
