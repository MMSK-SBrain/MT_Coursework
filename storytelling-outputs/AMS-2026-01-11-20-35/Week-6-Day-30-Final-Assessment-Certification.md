**Sample Question (Scenario-Based):**

*Customer complains of spongy brake pedal after brake pad replacement. Pedal travel excessive, but brakes still function. What is the most likely cause and corrective action?*

A) Brake pads installed incorrectly; reinstall properly
B) Air in brake lines; bleed brake system
C) Master cylinder failed; replace master cylinder
D) Brake fluid contaminated; flush system

**Correct Answer:** B (Air entered system during caliper work; bleeding removes air, firms pedal)

### Part 3: Practical Assessment Details (45 minutes)

**Task 1: Precision Measurement (45 minutes, 20 points)**

**Objective:** Demonstrate measurement competency

**Assignment:**
- Measure cylinder bore (4 cylinders, 3 heights, 2 orientations = 24 measurements)
- Calculate taper and out-of-round
- Measure crankshaft journals using micrometer
- Calculate bearing clearance using Plastigage
- Measure piston-to-bore clearance
- Recommend repair action (hone, rebore 0.25mm, rebore 0.50mm)

**Evaluation Criteria:**
- Measurement accuracy (±0.01mm)
- Correct tool usage (bore gauge calibration, micrometer technique)
- Accurate calculations
- Correct diagnosis and recommendation

---

**Task 2: Diagnostic Problem-Solving (60 minutes, 25 points)**

**Objective:** Diagnose multi-system fault

**Scenario:**
- Vehicle: Honda City 1.5L (BS6)
- Complaint: Check engine light on, rough idle, poor fuel economy
- Tools Provided: OBD-II scan tool, multimeter, service manual

**Required Actions:**
1. Read DTCs and freeze frame data
2. Identify affected system(s)
3. Test relevant components (O2 sensors, MAF sensor, etc.)
4. Analyze fuel trim data
5. Diagnose root cause
6. Recommend repair with cost estimate
7. Document findings professionally

**Evaluation Criteria:**
- Systematic diagnostic approach (6-step methodology)
- Correct use of scan tool
- Accurate component testing
- Root cause identification (not just symptoms)
- Professional documentation

---

**Task 3: Repair Execution (90 minutes, 25 points)**

**Objective:** Perform complete repair with proper procedure

**Assignment (Randomly Selected from Pool):**
- Option A: Engine timing belt replacement (including timing marks, tensioner adjustment)
- Option B: Clutch replacement (including alignment, torque procedures)
- Option C: Brake caliper rebuild and bleeding
- Option D: Wheel alignment (camber, caster, toe adjustment)

**Evaluation Criteria:**
- Correct disassembly sequence
- Proper tool usage
- Torque specifications followed
- Safety procedures observed
- Component installation orientation correct
- Quality verification (test drive, measurements)

---

**Task 4: Service Documentation (30 minutes, 15 points)**

**Objective:** Complete professional service records

**Assignment:**
- Create detailed service report for Task 3 repair
- Include: Customer information, vehicle details, diagnosis, parts used, labor time, cost breakdown
- Multi-point inspection checklist
- Service recommendations for customer
- Quality assurance sign-off

**Evaluation Criteria:**
- Completeness (all fields filled accurately)
- Clarity (customer can understand)
- Professionalism (grammar, formatting)
- Accurate cost calculation
- Safety recommendations included

---

**Task 5: Customer Interaction (15 minutes, 15 points)**

**Objective:** Demonstrate professional communication

**Scenario:**
- Evaluator role-plays customer
- You must explain diagnosis from Task 2
- Present repair options with costs
- Answer customer questions
- Handle objections ("Are you sure it's not something cheaper?")

**Evaluation Criteria:**
- Clear explanation (avoids excessive jargon)
- Confidence and professionalism
- Honesty (doesn't oversell or understate)
- Handles objections calmly
- Builds trust

## Assessment Preparation Tips

**Theory Exam Tips:**
1. Review notes from all 30 days
2. Focus on key specifications (torque values, clearances, fluid specs)
3. Understand concepts, not just memorize
4. Practice scenario-based questions

**Practical Exam Tips:**
1. Read instructions carefully before starting
2. Follow systematic procedures (no shortcuts)
3. Double-check torque values
4. Document as you work (don't rely on memory)
5. Ask clarifying questions if needed

**Customer Interaction Tips:**
1. Speak clearly, avoid excessive jargon
2. Use visual aids (show scan tool data, parts)
3. Be honest about costs and timelines
4. Build trust through professionalism
5. Listen to customer concerns
