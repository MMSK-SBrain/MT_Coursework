# Requirements: IMSF Assessment Suite

**Defined:** 2026-02-17
**Core Value:** Facilitators can confidently measure, evaluate, and evidence behavioral change in male shop floor managers through assessment instruments grounded in behavioral science and tailored to the Indian manufacturing context.
**Research basis:** LIKERT-DESIGN.md, SCENARIO-MCQ-DESIGN.md, ACTION-PLAN-DESIGN.md, PITFALLS.md

---

## v1.1 Requirements

8 printable documents across 3 assessment instruments. Documents 1 & 7 are the same physical form (Pre/Post columns); listed separately for phase planning.

---

### Instrument A: Pre/Post Self-Assessment Form

*Participant-facing. Completed twice: Pre at 9 AM (before content), Post at 4:45 PM (end of day).*

- [ ] **FORM-01**: Form contains exactly 21 behavioral items, 3 per each of the 7 course domains (Foundation, Daily Interactions, Self-Awareness, Empathy, Feedback & Conflict, Health Sensitivity, Team Inclusion)
- [ ] **FORM-02**: Items use a 5-point frequency scale with anchors: 1=Almost Never, 2=Rarely, 3=Sometimes, 4=Often, 5=Almost Always
- [ ] **FORM-03**: All items are written in first-person, present-tense, behavioral language describing observable actions (not attitudes or intentions)
- [ ] **FORM-04**: Items use growth framing (positive behaviors to aspire to), not deficit framing (negative behaviors to stop)
- [ ] **FORM-05**: Item framing ratio is approximately 60% universal ("all team members") and 40% gender-specific ("women on my team")
- [ ] **FORM-06**: Exactly 3 items are reverse-scored, distributed across the form (one in each third); reverse-scored items are NOT marked on the participant form
- [ ] **FORM-07**: Health Sensitivity domain items use health-general language ("appears unwell," "in discomfort") rather than explicitly naming menstruation
- [ ] **FORM-08**: Domain order follows the prescribed sequence: Foundation → Daily Interactions → Self-Awareness → Empathy → Feedback & Conflict → Health Sensitivity → Team Inclusion
- [ ] **FORM-09**: Domain sections are separated by visual breaks (horizontal rule or whitespace) with NO section labels on the participant form
- [ ] **FORM-10**: Form has two named columns — "Pre (AM)" and "Post (PM)" — on the same physical sheet; Post column has light shading to distinguish it visually
- [ ] **FORM-11**: Pre-column responses remain visible when participant completes the Post column (deliberate — the visible delta is the reflection)
- [ ] **FORM-12**: Form is titled "My Management Practices" with subtitle "A personal reflection tool — for your eyes only"; words "assessment," "evaluation," or "test" do not appear on the participant form
- [ ] **FORM-13**: Header states explicitly: "This form is not collected by the facilitator. It is yours to keep." Name field is marked "(optional)"
- [ ] **FORM-14**: Bottom of form includes a 3-question Self-Reflection guide: (1) Which domain showed your biggest shift? (2) Which item surprised you most? (3) What ONE behavior will you focus on first?
- [ ] **FORM-15**: Optional domain-averaging grid is included at the bottom for participants who want to calculate domain scores

### Instrument A: Self-Assessment Facilitator Administration Guide

*Facilitator-facing. Covers administration of both Pre and Post assessments.*

- [ ] **GUIDE-SA-01**: Guide maps each of the 21 items to its domain by item number
- [ ] **GUIDE-SA-02**: Reverse-scored items are clearly marked with (R); a conversion table is provided (1→5, 2→4, 3→3, 4→2, 5→1)
- [ ] **GUIDE-SA-03**: Verbatim facilitator script is provided for introducing the Pre-assessment (including the privacy guarantee, instruction to rate actual behavior not intentions, 5-minute time allocation)
- [ ] **GUIDE-SA-04**: Verbatim facilitator script is provided for introducing the Post-assessment (including instruction to rate "right now" not based on morning responses)
- [ ] **GUIDE-SA-05**: Aggregate data collection method is specified: show-of-hands protocol with exact questions the facilitator can ask (e.g., "Raise your hand if you noticed a shift of 2+ points in any domain")
- [ ] **GUIDE-SA-06**: Optional anonymous tear-off strip design is described (total pre-score, total post-score, largest single-item shift) as an alternative to show-of-hands
- [ ] **GUIDE-SA-07**: Cultural context notes are provided for each sensitive domain (Health Sensitivity, Team Inclusion) — how to handle defensive reactions, what language to use
- [ ] **GUIDE-SA-08**: Guide explicitly states: facilitator does NOT collect participant forms; forms go home with participants

### Instrument B: Scenario-Based Assessment Booklet

*Participant-facing. Completed individually during Module 6 (3:45–4:45 PM).*

- [ ] **SCEN-01**: Booklet contains exactly 5 scenarios in the prescribed sequence: (1) Daily greeting/interaction, (2) Health accommodation, (3) Professional feedback to woman employee, (4) Male peer behaving rudely to woman employee, (5) Fair task allocation
- [ ] **SCEN-02**: Each scenario vignette is 60-90 words and follows the 3-part structure: Setting (who/where) + Context (what has been happening) + Trigger (what just happened requiring a response)
- [ ] **SCEN-03**: Each scenario uses authentic Indian manufacturing context: real job titles (Shift Supervisor, Section In-Charge, Press Operator), real workplace language (toolbox talk, shift handover, canteen), real Indian names (pan-regional)
- [ ] **SCEN-04**: Each scenario presents exactly 4 answer options (A, B, C, D) graded on the Best/Adequate/Poor/Harmful scale
- [ ] **SCEN-05**: Options are written using behavioral gradation: Best (3pts) = inclusive and professionally skilled; Adequate (2pts) = good intent but incomplete; Poor (1pt) = well-intentioned but counterproductive; Harmful (0pts) = normalised pre-training behaviour
- [ ] **SCEN-06**: The Harmful option for each scenario is NOT cartoonishly bad — it represents a plausible "business as usual" response, not obvious cruelty
- [ ] **SCEN-07**: All four options begin with an action verb and are approximately equal in length (15-30 words each)
- [ ] **SCEN-08**: Best answer position is randomised across the 5 scenarios (no more than 2 consecutive scenarios with the best answer in the same position; all 4 positions A/B/C/D used across the set)
- [ ] **SCEN-09**: Question stem for every scenario is identical in structure: "As [specific role], what is the BEST course of action?"
- [ ] **SCEN-10**: Scenario difficulty follows the prescribed sequence: Easy → Easy-Medium → Medium → Medium-Hard → Medium (warm-up to consolidation arc)
- [ ] **SCEN-11**: Booklet includes an instructions page explaining: individual work, 15-minute time limit, BEST answer not "correct" answer, and that group debrief follows
- [ ] **SCEN-12**: A detachable or separate Answer Sheet is included with: rows for scenarios 1-5 (Your Answer + Points columns), a Total line, and 4 score interpretation bands (13-15, 9-12, 5-8, 0-4) with non-judgmental framing language
- [ ] **SCEN-13**: Answer Sheet states explicitly: "This score is for YOUR reflection only. It is not shared with your company or HR."

---

### Instrument B: Scenario Answer Key and Debrief Facilitation Guide

*Facilitator-facing. Used during Module 6 group debrief (40 minutes across 5 scenarios).*

- [ ] **GUIDE-SC-01**: For each scenario, the guide specifies the best answer letter and total point value for each option (3/2/1/0)
- [ ] **GUIDE-SC-02**: For each scenario, the guide provides a 3-4 sentence rationale for WHY the Best answer is best, connecting back to specific course concepts from the day
- [ ] **GUIDE-SC-03**: For each scenario, the guide explains why each other option falls short: Adequate ("reasonable but misses..."), Poor ("good intent but causes..."), Harmful ("normalised behaviour that we are changing because...")
- [ ] **GUIDE-SC-04**: Each scenario entry includes one "Key Teaching Point" sentence per non-best option — a single memorable line the facilitator can deliver verbally
- [ ] **GUIDE-SC-05**: Each scenario entry includes 2-3 discussion questions for the debrief, categorised: (1) personal reflection, (2) connects to real workplace, (3) challenges the Adequate answer specifically
- [ ] **GUIDE-SC-06**: Each scenario entry includes a "Watch For in the Room" section listing: one defensive behaviour to anticipate with redirect strategy, one common justification to listen for, and one breakthrough signal
- [ ] **GUIDE-SC-07**: Each scenario entry includes "Facilitator Notes" for cultural sensitivity, things to avoid saying, and how to handle the most likely objection
- [ ] **GUIDE-SC-08**: Guide specifies time allocation per scenario debrief: Scenarios 1 and 5 = 6 min each; Scenarios 2 and 4 = 10 min each; Scenario 3 = 8 min; Total = 40 min
- [ ] **GUIDE-SC-09**: Guide includes the execution model: Phase 1 Individual (all 5 at once, 15 min) → Phase 2 Sequential Debrief (40 min) → Phase 3 Score & Reflect (5 min)
- [ ] **GUIDE-SC-10**: Guide includes the self-scoring procedure: facilitator announces points per option for each scenario; participants self-score; NO public announcement of individual scores
- [ ] **GUIDE-SC-11**: Guide includes four named facilitation techniques: Show of Hands (before revealing answer), Devil's Advocate Reframe, Real Story Bridge, No Shame Frame — with example language for each
- [ ] **GUIDE-SC-12**: Health accommodation scenario (Scenario 2) includes a note on how to handle the question "But how do we know it's her period?" with exact facilitator response

### Instrument C: Personal Action Plan

*Participant-facing. Completed during Module 7 (4:45–5:00 PM). Participant keeps the form.*

- [ ] **PLAN-01**: Form contains all 7 sections in sequence: Header/Intent Statement, Reflection Anchor, Three Commitment Slots, Behavioral Coverage Check, Commitment Circle Preparation, Peer Commitment Recording Table, Two-Week Self-Check
- [ ] **PLAN-02**: Header states explicitly that the plan is not submitted to HR or management; "MY PERSONAL ACTION PLAN" with possessive framing
- [ ] **PLAN-03**: Reflection Anchor (Section 1) prompts: "Think of ONE specific moment in the past month when you could have handled an interaction with a woman colleague differently" with two writing lines
- [ ] **PLAN-04**: Each of the three Commitment Slots contains exactly 4 sub-fields: (1) The situation I am addressing, (2) When this situation happens, I will, (3) Instead of my old pattern of, (4) I will know I am succeeding when
- [ ] **PLAN-05**: The commitment writing formula is printed at the top of Section 2: "When [specific situation], I will [specific action] instead of [old habit]" with one model example
- [ ] **PLAN-06**: Three model example commitments are printed on the form covering different behavioral areas: health accommodation, feedback delivery, and task allocation — using authentic Indian workplace language
- [ ] **PLAN-07**: An optional Bonus Commitment slot is included (less structured, for participants who finish early)
- [ ] **PLAN-08**: Behavioral Coverage Check (Section 3) lists the 7 behavioral areas from the course as checkboxes with a note to aim for 2-3 different areas across commitments
- [ ] **PLAN-09**: Commitment Circle Preparation (Section 4) includes: instruction to choose the most personally meaningful commitment (not the easiest), a space to note which commitment is selected, and a one-sentence verbal formula "Starting tomorrow, I will..."
- [ ] **PLAN-10**: Peer Commitment Recording Table (Section 5) has 5 rows (Name + Their Commitment)
- [ ] **PLAN-11**: Two-Week Self-Check (Section 6) includes: a blank for writing the follow-up date, a 3-row table (Commitment # | Yes/Partly/No | What happened?), and two reflection prompts (if followed through / if not followed through)
- [ ] **PLAN-12**: Form includes a phone calendar reminder prompt: "Put this date in your phone calendar now: ____________"
- [ ] **PLAN-13**: A perforated wallet-card tear-off is included at the bottom: "Starting tomorrow, I will: _______" with signature and date line
- [ ] **PLAN-14**: Form closes with a commitment statement and signature line: "I, ______, choose to be a more inclusive leader. I make these commitments not because I was told to, but because I want my team to thrive."
- [ ] **PLAN-15**: Physical format: A4 double-sided, 120gsm or heavier paper; professional design (no clip art, no worksheety appearance); adequate line height for handwriting

### Instrument C: Action Plan Evaluation Rubric and Facilitator Guide

*Facilitator-facing. Used during Module 7 while circulating among participants.*

- [ ] **GUIDE-AP-01**: Rubric evaluates commitments on exactly two dimensions: Specificity (Can I picture this happening?) and Depth (Does this reflect real self-awareness?) — each with 3 rating levels (Strong/Developing/Vague)
- [ ] **GUIDE-AP-02**: Each rating level includes: a description of what it means, and observable indicators the facilitator looks for on the participant form
- [ ] **GUIDE-AP-03**: Coaching Decision Matrix covers all combinations of Specificity + Depth ratings with prescribed facilitator actions (Affirm / Light nudge / Coach specificity / Coach both / Direct intervention)
- [ ] **GUIDE-AP-04**: Coaching question sequence for vague commitments is provided in escalating order: (1) "Tell me about this one," (2) "When did this last happen?", (3) "What would I see you doing differently?", (4) "Can you rewrite this as: When [situation], I will [action]?"
- [ ] **GUIDE-AP-05**: Tone guidance is explicit: facilitator crouches/sits beside participant; uses "this is a good start" before any improvement suggestion; never says "too vague"; coaching = questions only, not content suggestions
- [ ] **GUIDE-AP-06**: Clear criteria for when to intervene vs. when to let participant struggle (blank form for 2+ minutes = intervene; slow thoughtful writing = leave alone)
- [ ] **GUIDE-AP-07**: Facilitator Quick-Scan Checklist (5 items to mentally check per participant) is provided for use during circulation
- [ ] **GUIDE-AP-08**: Commitment Circle facilitation script is included: facilitator opening statement, round format description, "Starting tomorrow, I will..." instruction, and closing statement
- [ ] **GUIDE-AP-09**: Module 7 timing script is provided with minute-by-minute breakdown (4:45–5:00 PM): introduction, reflection anchor, writing time, coverage check, phone calendar reminder, Commitment Circle, closure
- [ ] **GUIDE-AP-10**: A Module 7 Facilitator Summary Sheet is included (separate document within the guide): aggregate quality ratings, behavioral areas selected, patterns observed, engagement level, Commitment Circle observations

### Cross-Cutting: Production Requirements

- [ ] **PROD-01**: All participant documents are formatted for A4 print, black-and-white compatible (no color dependency for core content)
- [ ] **PROD-02**: All documents use 11pt body font minimum with adequate line spacing (8mm minimum) for handwriting in fields
- [ ] **PROD-03**: Participant documents use professional design language appropriate for engineering managers — no clip art, no "worksheety" appearance, no infantilising visual elements
- [ ] **PROD-04**: Facilitator guides are clearly distinguished from participant documents — different header design, explicit label "FACILITATOR GUIDE — NOT FOR DISTRIBUTION"
- [ ] **PROD-05**: All documents are produced as Word (.docx) files for editability, with a print-ready PDF version of each
- [ ] **PROD-06**: All documents are stored in `courses/inclusive-management-shop-floor/assessments/` with clear naming convention: `[instrument]-[audience].docx` (e.g., `pre-post-self-assessment-participant.docx`, `scenario-assessment-facilitator-guide.docx`)

## Out of Scope

| Feature | Reason |
|---------|--------|
| 30/60/90-day follow-up instruments | Deferred to v1.2 to keep v1.1 focused on training-day experience |
| Digital/online form versions | Printable-only for v1.1; digital is a separate design challenge |
| Hindi translations | English-only per project constraints; facilitator provides verbal Hindi clarification |
| Grading/scoring for HR records | Participant scores are private; no accountability data flows to HR |
| Video or interactive assessment formats | Out of scope for document-based milestone |
| Assessments for other courses | IMSF-specific only; generalization is a separate feature |
