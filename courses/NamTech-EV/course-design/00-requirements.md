# Course Requirements: NamTech EV Lab Course

**Created:** 2026-01-06
**Status:** Gathering
**Course Type:** Practical Lab Course (Experiment-Based)

## 1. Course Context
- **Topic:** Electric Vehicle Systems - Hands-on Laboratory Training
- **Industry/Domain:** Automotive - Electric Vehicles
- **Duration:** Modular/Flexible (self-paced or semester-based)
- **Structure:** 10 major equipment-based modules, each with multiple experiments
- **Description:** Practical lab course focused on hands-on experiments with EV components, systems, and vehicles. All learning outcomes are experiment-based using 10 major equipment setups. Designed to prepare engineering students for industry jobs by providing hands-on competencies in battery systems, motor control, charging infrastructure, vehicle integration, and testing.

## 2. Equipment Available (10 Major Setups)

### Setup 1: Battery Pack Prototyping and Manufacturing
- Complete assembly & testing setup
- Practice set for assembly/dismantling
- Smart BMS (2 sets)
- Spot-welding machine, IR Tester
- Workstation with tools

**Possible Experiments:**
- Battery pack assembly (series/parallel configurations)
- Spot welding practice
- BMS integration and testing
- Cell testing and selection
- Safety testing procedures

### Setup 2: BMS CAN Protocol cum Analyzer
- CAN communication and data logging
- Programmable BMS system
- Load testing system
- Thermal compensation features

**Possible Experiments:**
- BMS programming and configuration
- CAN protocol analysis
- State of Charge (SoC) monitoring
- Cell balancing experiments
- Temperature protection testing
- Load application and monitoring

### Setup 3: Battery Pack Charging and Discharging Test Bench
- Voltage range: 12V-84V
- Charge/discharge cycling (up to 16 cycles)
- Data logging and analysis

**Possible Experiments:**
- Charge-discharge cycle testing
- Capacity measurement
- Protection voltage verification
- Aging/cycle life testing
- Performance analysis under different loads

### Setup 4: CAN-LIN System Rig (Hydraulic Dynamometer)
- 1 kW PMSM and BLDC motors
- Hydraulic dyno with torque/speed measurement
- MATLAB/Simulink support
- FOC and six-step commutation algorithms

**Possible Experiments:**
- Motor performance testing (torque-speed curves)
- FOC algorithm implementation
- Efficiency mapping
- Load testing with hydraulic dyno
- Motor controller tuning
- CAN communication between controllers

### Setup 5: Vehicle Control Unit (VCU) & Algorithm Developer
- NXP automotive-grade microcontroller
- MATLAB/Simulink integration
- Mini HIL setup
- CAN, UART, SPI interfaces

**Possible Experiments:**
- VCU logic development
- Control algorithm programming
- HIL testing and validation
- CAN communication programming
- Automotive lighting control
- Functional safety testing

### Setup 6: Electric Vehicle Charging Simulator (CCS2)
- ISO 15118 protocol simulation
- Grid impact analysis
- Security and TLS features

**Possible Experiments:**
- CCS charging protocol analysis
- Message sequence verification
- Error injection and handling
- Grid load flow analysis
- Smart charging algorithms
- Harmonic analysis (THD)

### Setup 7: 4-Wheeler Drive Train with MIDC Cycle
- 2 kW PMSM motor
- Regenerative braking system
- MIDC cycle simulation
- Complete drivetrain integration

**Possible Experiments:**
- MIDC driving cycle simulation
- Regenerative braking efficiency testing
- Energy recovery measurement
- Drivetrain efficiency analysis
- Thermal management testing
- Fault diagnosis and troubleshooting

### Setup 8: Electric 2W Assembly Teardown Setup
- Complete e-scooter with 1kW BLDC hub motor
- Assembly/dismantling tools
- Hydraulic lift
- Special tools kit

**Possible Experiments:**
- Complete vehicle assembly/disassembly
- Component identification and function
- Wiring harness installation
- Torque specifications practice
- Safety procedures and PPE
- Diagnostic procedures

### Setup 9: Hybrid Vehicle Control Strategy Test Platform
- Honda Activa 5G based hybrid
- 1 kW BLDC motor + IC engine
- 48V 35Ah Li-ion battery
- Automatic changeover at 20 km/h

**Possible Experiments:**
- Hybrid mode switching analysis
- EV to IC transition testing
- Regenerative braking in hybrid mode
- Energy management strategy evaluation
- Speed-based control logic testing
- Fuel efficiency vs electric efficiency comparison

### Setup 10: E-Vehicle Research Lab Car
- 19 kW / 29.5 kW PMSM motor
- Open-source controller with Simulink support
- 12.8 kWh LiFePO4 battery
- Full BMS integration with CAN

**Possible Experiments:**
- FOC algorithm development and testing
- BMS-Motor controller integration
- Real vehicle performance testing
- SoC and SoH monitoring
- Regenerative braking optimization
- Complete powertrain calibration
- On-road performance validation

## 3. Target Audience
- **Primary Learners:** Undergraduate (UG) and Postgraduate (PG) engineering students
- **Experience Level:** Engineering students with foundational knowledge in electrical, electronics, and mechanical systems
- **Prerequisites:**
  - Basic electrical circuits and electronics
  - Understanding of DC/AC motors
  - Basic programming knowledge (C/MATLAB preferred)
  - Automotive systems fundamentals
- **Job Role:** Engineering positions in EV industry (Design Engineer, Test Engineer, R&D Engineer, Integration Engineer, Application Engineer)

## 4. Learning Outcomes Framework
All learning outcomes will be:
- **Action-oriented** (using practical verbs: assemble, test, measure, diagnose, configure, calibrate, etc.)
- **Experiment-based** (tied to specific equipment and hands-on activities)
- **Performance-focused** (students will demonstrate competency through practical tasks)
- **Safety-integrated** (safe operation of equipment is paramount)

## 5. Course Structure Decision

**Chosen Approach:** Option B - System-by-System (Functional Grouping)

**Module Organization:**

### Module 1: Battery Systems
- Setup 1: Battery Pack Prototyping and Manufacturing
- Setup 2: BMS CAN Protocol cum Analyzer
- Setup 3: Battery Pack Charging and Discharging Test Bench

### Module 2: Motor & Drivetrain Systems
- Setup 4: CAN-LIN System Rig (Hydraulic Dynamometer)
- Setup 7: 4-Wheeler Drive Train with MIDC Cycle and Regenerative System

### Module 3: Charging Infrastructure
- Setup 6: Electric Vehicle Charging Simulator (CCS2)

### Module 4: Vehicle Control Systems
- Setup 5: Vehicle Control Unit (VCU) & Algorithm Developer
- Setup 9: Hybrid Vehicle Control Strategy Test Platform

### Module 5: Complete Vehicle Integration
- Setup 8: Electric 2W Assembly Teardown Setup
- Setup 10: E-Vehicle Research Lab Car

## 6. Assessment Strategy
- No formal certification required
- Performance-based competency demonstration through experiments
- Lab reports and experiment completion tracking (institutional discretion)
