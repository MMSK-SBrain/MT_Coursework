## Recording Strategy Guide

### Using Quiz Questions in Videos

**During Lectures:**
- Reference quiz questions as "checkpoints"
- "Here's a question you'll see on the quiz..."
- Build anticipation for assessment
- Show which concepts are most important

**Examples:**
```
"Now, on the quiz you'll be asked:
What's the difference between classification and regression?

Let me show you with this example..."
```

### Using Rubrics in Demonstrations

**When Showing Projects:**
- Walk through rubric criteria
- "To get full points on this section..."
- Show "excellent" level work
- Point out common mistakes
- Build toward clear success

**Examples:**
```
"Looking at the rubric, you need 5+ visualizations
for excellent. Let me show you what that looks like..."
```

### Structuring Session Videos

**Format per Session:**
1. **Introduction (2 min)**
   - Session overview
   - Learning objectives (from quiz topics)

2. **Concept Teaching (15-25 min)**
   - Core content
   - Reference quiz questions

3. **Live Coding Demo (20-30 min)**
   - Follow rubric criteria
   - Show excellent-level work

4. **Common Mistakes (5-10 min)**
   - From rubric "poor" level
   - How to avoid them

5. **Recap & Next Steps (2-3 min)**
   - Review objectives
   - Preview next session

**Total:** 45-70 minutes per session
