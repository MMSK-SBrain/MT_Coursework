data['MA_10'] = data['Close'].rolling(window=10).mean()
data['MA_50'] = data['Close'].rolling(window=50).mean()
data['Returns'] = data['Close'].pct_change()
data['Volatility'] = data['Returns'].rolling(window=10).std()

# RSI Calculation
def calculate_rsi(data, window=14):
    delta = data['Close'].diff()
    gain = (delta.where(delta > 0, 0)).rolling(window).mean()
    loss = (-delta.where(delta < 0, 0)).rolling(window).mean()
    rs = gain / loss
    rsi = 100 - (100 / (1 + rs))
    return rsi

## 🎓 Pedagogical Framework

### The Corrected Progression (After Analysis)

**Original Problem:**
- Module 1 had learners building models before teaching them

**Solution:**
```
❌ BUILD → UNDERSTAND (wrong order)
✅ HOOK → UNDERSTAND → EXPLORE → BUILD (correct!)
```

**The 6-Phase Learning Model:**

1. **HOOK** (Module 0)
   - Emotion: Wonder
   - Activity: Watching demos
   - Outcome: "I want this!"

2. **UNDERSTAND** (Module 1)
   - Emotion: Clarity
   - Activity: Conceptual learning
   - Outcome: "I get it now!"

3. **EXPLORE** (Module 1)
   - Emotion: Discovery
   - Activity: Data exploration
   - Outcome: "I found patterns!"

4. **BUILD** (Modules 2-3)
   - Emotion: Accomplishment
   - Activity: Creating models
   - Outcome: "I made it work!"

5. **OPTIMIZE** (Modules 4-9)
   - Emotion: Mastery
   - Activity: Improving and deploying
   - Outcome: "I made it better!"

6. **CREATE** (Module 10)
   - Emotion: Pride
   - Activity: Original work
   - Outcome: "This is MY creation!"

### Cognitive Load Management

**LOW Load Modules (1 week):**
- Module 0: Watching and setup
- Module 1: Understanding and exploring

**MEDIUM Load Modules (1.5 weeks):**
- Module 4: Optimization (building on existing knowledge)

**HIGH Load Modules (2 weeks):**
- Module 2: First building (new skill)
- Module 3: Regression (complex math)
- Module 5: Unsupervised (new paradigm)
- Module 6: Neural networks (complexity)
- Module 7: Computer vision (new domain)
- Module 8: NLP (new domain)
- Module 9: Deployment (production complexity)

**INTENSIVE Module (3 weeks):**
- Module 10: Capstone (integration of everything)

### Assessment Philosophy

**Formative Assessment (Throughout):**
- Quizzes after each session (immediate feedback)
- Hands-on activities (practice with feedback)
- AI-assisted debugging (learning from errors)
