## Account Creation

### Step 1: Navigate to ChatGPT

1. Open your web browser
2. Go to: **https://chat.openai.com**
3. You'll see the ChatGPT landing page

**Screenshot placeholder:** *ChatGPT landing page with "Sign up" button*

### Step 2: Click "Sign up"

1. Click the **"Sign up"** button (usually green/black)
2. You'll be presented with signup options:
   - Continue with Google
   - Continue with Microsoft Account
   - Continue with Apple
   - Continue with email

**Screenshot placeholder:** *Sign up options page*

### Step 3: Choose Signup Method

**Option A: Email Signup (Recommended for beginners)**

1. Click **"Continue with email"**
2. Enter your email address
3. Click **"Continue"**
4. Create a strong password (min 8 characters)
5. Click **"Continue"**

**Screenshot placeholder:** *Email signup form*

**Option B: Google Signup (Faster)**

1. Click **"Continue with Google"**
2. Select your Google account
3. Grant permissions when prompted
4. Done!

**Screenshot placeholder:** *Google OAuth screen*

### Step 4: Verify Your Email

1. Check your email inbox
2. Look for email from "OpenAI" (check spam if not found)
3. Click the verification link
4. You'll be redirected back to ChatGPT

**Screenshot placeholder:** *Verification email example*

### Step 5: Complete Profile

1. Enter your name (first and last)
2. (Optional) Enter phone number
3. Accept Terms of Service
4. Click **"Continue"**

**Screenshot placeholder:** *Profile completion form*

### Step 6: Welcome to ChatGPT!

You'll see a welcome screen explaining:
- What ChatGPT can do
- Examples of questions you can ask
- Limitations to be aware of

Click **"Next"** through the tutorial screens.

**Screenshot placeholder:** *Welcome tutorial screens*

## Troubleshooting

### Common Issues and Solutions

#### Issue 1: "Sign up button doesn't work"

**Solutions:**
- Clear browser cache and cookies
- Try different browser (Chrome recommended)
- Disable ad blockers
- Check internet connection

#### Issue 2: "Verification email not received"

**Solutions:**
- Check spam/junk folder
- Wait 5-10 minutes
- Check email address spelling
- Click "Resend verification email"
- Try different email provider

#### Issue 3: "ChatGPT is at capacity"

**Solutions:**
- Wait 10-15 minutes and try again
- Try during off-peak hours (early morning)
- Refresh the page
- Consider ChatGPT Plus (paid) - NOT required for course

#### Issue 4: "Responses are slow or incomplete"

**Solutions:**
- Check internet connection
- Close unnecessary browser tabs
- Try during off-peak hours
- If response cuts off, type "continue" to get more

#### Issue 5: "Can't access from my country"

**Solutions:**
- Check [OpenAI availability list](https://openai.com/blog/chatgpt-available-countries)
- Use VPN (if allowed in your region)
- Contact course support for alternatives (Gemini, Claude)

## Tips for Course

### Best Practices for ML Learning

#### 1. **Structure Your Prompts**

**Good:**
```
Explain decision trees in machine learning.
Include: definition, how they work, and one example.
```

**Bad:**
```
decision trees?
```

#### 2. **Ask for Code with Explanations**

```
Write Python code to load a CSV file and display first 5 rows.
Include comments explaining each line.
```

#### 3. **Request Beginner-Friendly Explanations**

```
Explain overfitting like I'm a beginner who just started learning ML.
Use a simple analogy.
```

#### 4. **Break Complex Questions into Steps**

Instead of:
```
How do I build a stock predictor?
```

Try:
```
1. What data do I need for a stock price predictor?
2. What ML algorithm is best for stock prediction?
3. Show me Python code to prepare stock data for ML.
```

#### 5. **Use Follow-up Questions**

```
First: Explain what pandas is
Follow-up: Now show me code to filter rows in pandas
Follow-up: What if I want to filter by multiple conditions?
```

### Keyboard Shortcuts

- **Enter**: Send message
- **Shift + Enter**: New line in message
- **Ctrl/Cmd + K**: New chat
- **Ctrl/Cmd + Shift + ;**: Copy last response

### Organizing Your Chats

1. **Rename chats** for easy reference:
   - Click chat in history
   - Click pencil icon
   - Name it: "Module 1 - Pandas Questions"

2. **Archive old chats**:
   - Click "..." next to chat
   - Select "Archive"

3. **Create folders** (ChatGPT Plus only)

## Free vs Paid Comparison

### ChatGPT Free (GPT-3.5)

✅ **Sufficient for this course**
- ✅ Unlimited messages
- ✅ Code generation
- ✅ Explanations
- ✅ Debugging help
- ✅ Standard response time
- ❌ No GPT-4 access
- ❌ No priority access during peak times
- ❌ No plugins

**Cost:** $0/month

### ChatGPT Plus (GPT-4)

🌟 **Better but not required**
- ✅ GPT-4 access (more accurate, better reasoning)
- ✅ Faster responses
- ✅ Priority access during peak times
- ✅ Plugins (web browsing, code interpreter)
- ✅ Higher usage limits

**Cost:** $20/month

**Course Recommendation:** Start with Free tier. Upgrade only if:
- You find GPT-3.5 insufficient for complex tasks
- You frequently encounter "at capacity" errors
- You want cutting-edge AI capabilities
