## Troubleshooting

### Common Issues and Solutions

#### Issue 1: "Can't access Gemini"

**Solutions:**
- Ensure you're using a Google account
- Check if Gemini is available in your country: [Supported countries list](https://support.google.com/gemini/answer/13594961)
- Try different browser
- Clear browser cache
- Use VPN if needed (check local regulations)

#### Issue 2: "Responses are blocked/filtered"

**Solutions:**
- Gemini has content filters for safety
- Rephrase your prompt more professionally
- Avoid sensitive topics
- For technical questions, be specific about context

**Example:**
❌ "How to hack code"
✅ "How to debug Python code errors"

#### Issue 3: "Can't see chat history"

**Solutions:**
- Check if "Gemini Apps Activity" is ON
- Sign in with correct Google account
- Activity may take few minutes to sync

#### Issue 4: "Error: Something went wrong"

**Solutions:**
- Refresh page (F5)
- Clear browser cache
- Try incognito mode
- Check internet connection
- Try again in 5 minutes

#### Issue 5: "Response cut off mid-sentence"

**Solutions:**
- Type "continue" to get rest of response
- Reduce prompt complexity
- Break into smaller questions

## Tips for Course

### Gemini vs ChatGPT: When to Use Which

**Use Gemini when:**
- ✅ You want Google Search integration
- ✅ Need fact-checking with sources
- ✅ Prefer Google ecosystem integration
- ✅ Want multiple response drafts

**Use ChatGPT when:**
- ✅ Need very detailed code explanations
- ✅ Want more conversational responses
- ✅ Prefer traditional chat interface

**Best Practice:** Use BOTH and compare!

### Best Prompts for Gemini

#### 1. **Fact-Based Questions**

Gemini excels at factual queries with verification:

```
What are the most popular machine learning algorithms in 2025?
Include real-world applications.
```

#### 2. **Code with Explanations**

```
Write Python code to:
1. Load a CSV file
2. Calculate mean of a column
3. Create a histogram

Add detailed comments and explain the logic.
```

#### 3. **Comparisons**

```
Compare supervised vs unsupervised learning.
Create a table showing:
- Definitions
- Use cases
- Example algorithms
- Pros and cons
```

#### 4. **Step-by-Step Guides**

```
Provide a step-by-step guide to:
Train a decision tree classifier on the Iris dataset using scikit-learn.

Include code, explanations, and expected output.
```

### Gemini-Specific Features

#### 1. **Export to Google Docs**

1. Click export icon after response

#### 2. **Share Conversation**

**Screenshot placeholder:** *Share feature*

#### 3. **Response Drafts**

Sometimes Gemini shows multiple response options:

1. Look for "View other drafts" dropdown
2. Click to see alternatives

## Gemini Free vs Gemini Advanced

### Gemini (Free)

✅ **Sufficient for this course**
- ✅ Unlimited conversations
- ✅ Google Search integration
- ✅ Code generation
- ✅ Multi-turn conversations
- ✅ Export features
- ❌ Limited to standard model
- ❌ Some capacity limits during peak

**Cost:** $0/month

### Gemini Advanced (Paid)

🌟 **Better but not required**
- ✅ Most capable Gemini model (1.5 Pro)
- ✅ Longer context window (1M tokens)
- ✅ Priority access
- ✅ Advanced coding capabilities
- ✅ Integration with Google Workspace

**Cost:** $19.99/month (part of Google One AI Premium)

**Course Recommendation:** Free version is perfect for learning!

## Privacy and Data

### What Gemini Knows

- Your Google account info (name, email)
- Your conversation history (if Activity is ON)
