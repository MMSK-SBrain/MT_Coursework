## Instructor Notes

**Grading Philosophy for Module 0:**
- This is a **confidence-building** exercise, not a technical assessment
- Grade generously - focus on effort and engagement
- Look for genuine curiosity, not perfect answers
- If student tried all demos and answered questions thoughtfully → FULL CREDIT

**Common Issues:**
- Demo links not working → Provide alternative or extend deadline
- Student missed 1 demo → Still passing if other work is complete
- Screenshots unclear → Accept description if technical issue

**When to Give Full Credit:**
- All demos attempted (even if 1 failed due to technical issues)
- All reflection questions show genuine thought
- Evidence of curiosity and engagement

**When to Request Revision:**
- Fewer than 3 demos attempted
- No reflection questions answered
- No evidence submitted
- Copy-pasted answers with no personal thought
